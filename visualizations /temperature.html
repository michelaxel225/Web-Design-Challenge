Latitude vs. Max Temperature


latitude and longitudinal coordinates were generated in this project using almost 600 coordinates from different cities. With these coordinates, the nearest city to the equator was found. Using the weather API from openweathermap.org, data were retrieved and placed into a pandas data frame.As expected, the weather becomes significantly warmer as one approaches the equator.
